FooTable Pagination Plugin

FooTable is a fantastic jQuery plugin that create searchable, sortable, responsive and themeable tables. When I began using FooTable it handled large amounts of data swiftly, but did not paginate. The software was written in a very extendable fashion so I submitted a pull request with the pagination plugin and demo, and received a thanks @awc737. I highly recommend FooTable for the easiest dynamic and responsive HTML and JavaScript tables.
